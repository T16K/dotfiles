## Scripts

### kitty

- Change to `kitty` directory:
```
$ cd scripts/kitty
```

- To install it, run:
```
$ ./install
```

- Run `kitty-cat` & select the right option:
```

### rofi

- Change to `rofi` directory:
```
$ cd scripts/rofi
```

- Run `setup.sh` & select your screen resolution:
```
[*] Installing Rofi Themes...

[*] Choose your screen resolution -
[1] 1920x1080
[2] 1366x768

[?] Select Option : 1

[*] Installing fonts...
[*] Creating a backup of your rofi configs...
[*] Successfully Installed.
```

### polybar

- Change to `polybar` directory:
```
$ cd scripts/polybar
```

- Run `setup.sh` & select a style:
```

[*] Choose Style -
[1] Simple
[2] Bitmap

[?] Select Option : 1

[*] Installing fonts...
